## UI 컴포넌트 명세

### 1. PropertyGallery

**역할**: 이미지 그리드 표시 + 라이트박스

**인터랙션**:
- 메인 이미지 클릭 → 라이트박스 (전체 화면 갤러리)
- "[📷 전체 사진 보기]" 버튼 → 라이트박스
- 라이트박스 내 좌우 화살표로 이미지 탐색
- ESC 또는 X 버튼으로 닫기

**반응형**:
- 데스크톱: 그리드 레이아웃 (1개 대형 + 4개 소형)
- 모바일: 캐러셀 (스와이프)

**Props**:
```tsx
interface PropertyGalleryProps {
  images: string[]; // 이미지 URL 배열
  alt: string; // 숙소 이름 (대체 텍스트)
}
```

---

### 2. BookingWidget (예약 위젯)

**역할**: 날짜 선택 + 가용성 확인 + 가격 계산 + 예약 버튼

**상태**:
- `checkIn`: Date | null
- `checkOut`: Date | null
- `guests`: number (기본값: 2)
- `isAvailable`: boolean
- `totalPrice`: number

**인터랙션**:
1. 체크인 날짜 선택 → 달력 팝오버
2. 체크아웃 날짜 선택 → 달력 팝오버 (체크인 이후만 선택 가능)
3. 날짜 선택 완료 → API 호출: `GET /api/availability/check`
4. 가용 시 → 가격 표시 + "예약하기" 버튼 활성화
5. "예약하기" 클릭 → `/booking/checkout?propertyId=...&checkIn=...&checkOut=...&guests=...`

**반응형**:
- 데스크톱: 우측 Sticky (스크롤 시 상단 고정)
- 모바일: 하단 고정 바 (Fixed Position)

**Props**:
```tsx
interface BookingWidgetProps {
  property: {
    id: string;
    name: string;
    pricePerNight: number;
    maxGuests: number;
  };
}
```

**API 호출**:
```typescript
// 가용성 확인
const checkAvailability = async () => {
  const response = await fetch(
    `/api/availability/check?propertyId=${propertyId}&checkIn=${checkIn}&checkOut=${checkOut}`
  );
  const data = await response.json();
  setIsAvailable(data.available);
  setTotalPrice(data.totalPrice);
};
```

### 3. PropertyInfo

**역할**: 숙소 기본 정보 표시

**표시 정보**:
- 숙소 이름
- 평점 + 리뷰 수 (⭐ 4.8 (32))
- 위치 (간단: "평창군")
- 숙소 유형, 최대 인원, 침실/침대/욕실 수

**레이아웃**:

- 이미지 포함 가능

## 접근성 (A11y)

- **키보드 네비게이션**:
  - Tab: 갤러리 → 예약 위젯 → 리뷰 순서
  - Space/Enter: 버튼 활성화
- **스크린 리더**:
  - 이미지 alt 텍스트: "논뷰맛집 한옥 스테이 - 메인 이미지"
  - 버튼 aria-label: "예약하기 - 1박당 50,000원"
- **컬러 대비**: WCAG AA 준수 (텍스트 4.5:1 이상)
- **포커스 표시**: 파란 아웃라인 (2px solid)

## 에러 상태

1. **숙소 없음** (404):
   - 중앙 카드 표시: "존재하지 않는 숙소입니다."
   - CTA: "숙소 탐색" 버튼 → `/explore`

2. **날짜 선택 안 됨**:
   - 예약 위젯: "날짜를 선택해주세요" (회색 텍스트)
   - "예약하기" 버튼 비활성화

3. **가용성 없음**:
   - 예약 위젯: "선택한 날짜에 예약할 수 없습니다."
   - 대안 날짜 제안 (향후)
