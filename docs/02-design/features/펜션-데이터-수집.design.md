## 3. Property 모델 업데이트 설계

### app/models/property.rb (수정 후)

```ruby
class Property < ApplicationRecord
  has_and_belongs_to_many :tags

  enum :status, { draft: 0, active: 1, archived: 2 }

  # 크롤링 데이터 여부 확인
  scope :auto_collected, -> { where(source: "uploaded") }
  scope :manual, -> { where(source: "manual") }
  scope :pending_review, -> { draft.auto_collected }

  # price_per_night는 크롤링 데이터에서 nil 허용 (관리자가 나중에 입력)
  validates :title, presence: true
  validates :price_per_night, numericality: { greater_than: 0 }, allow_nil: true
end
```

## 4. 서비스 클래스 설계

### 4.1 BaseCrawler (공통 인터페이스)

**파일**: `app/services/crawlers/base_crawler.rb`

```ruby
module Crawlers
  class BaseCrawler
    RATE_LIMIT_SECONDS = 1  # API 요청 간 딜레이

    def call
      raise NotImplementedError, "#{self.class}#call 구현 필요"
    end

    private

    def log(message)
      logger = Logger.new(Rails.root.join("log/crawl.log"))
      logger.info("[#{self.class.name}] #{message}")
      Rails.logger.info("[CRAWL] #{message}")
    end

    def sleep_rate_limit
      sleep(RATE_LIMIT_SECONDS)
    end
  end
end
```

### 4.2 TourApiCrawler (한국관광공사 TourAPI)

**파일**: `app/services/crawlers/tour_api_crawler.rb`

```ruby
module Crawlers
  class TourApiCrawler < BaseCrawler
    BASE_URL = "https://apis.data.go.kr/B551011/KorService1/areaBasedList1"
    CONTENT_TYPE_LODGING = "32"  # 숙박 시설
    PAGE_SIZE = 100

    # 농촌 관련 지역 코드 (전국 주요 농촌 지역)
    AREA_CODES = {
      "강원" => "32",
      "충북" => "33",
      "충남" => "34",
      "전북" => "37",
      "전남" => "38",
      "경북" => "35",
      "경남" => "36",
      "제주" => "39"
    }

    def initialize(api_key: ENV["TOUR_API_KEY"])
      @api_key = api_key
    end

    def call
      results = []

      AREA_CODES.each do |area_name, area_code|
        log("#{area_name} 지역 수집 시작 (areaCode: #{area_code})")
        page_results = fetch_area(area_code)
        results.concat(page_results)
        log("#{area_name} 지역 #{page_results.size}건 수집 완료")
        sleep_rate_limit
      end

      log("전체 수집 완료: #{results.size}건")
      results
    end

    private

    def fetch_area(area_code, page_no: 1)
      uri = URI(BASE_URL)
      uri.query = URI.encode_www_form(
        serviceKey: @api_key,
        numOfRows: PAGE_SIZE,
        pageNo: page_no,
        MobileOS: "ETC",
        MobileApp: "VINTEE",
        _type: "json",
        listYN: "Y",
        arrange: "A",
        contentTypeId: CONTENT_TYPE_LODGING,
        areaCode: area_code
      )

      response = Net::HTTP.get_response(uri)
      return [] unless response.is_a?(Net::HTTPSuccess)

      data = JSON.parse(response.body)
      items = data.dig("response", "body", "items", "item") || []
      items = [items] unless items.is_a?(Array)  # 단일 결과 배열 처리

      items.map { |item| normalize(item) }
    rescue => e
      log("ERROR fetch_area(#{area_code}): #{e.message}")
      []
    end

    def normalize(item)
      {
        external_id:   item["contentid"],
        title:         item["title"],
        description:   nil,  # detail API 별도 호출 필요
        location:      [item["addr1"], item["addr2"]].compact.join(" ").strip,
        thumbnail_url: item["firstimage"].presence,
        latitude:      item["mapy"]&.to_f,
        longitude:     item["mapx"]&.to_f,
        phone:         item["tel"].presence,
        source:        "uploaded",
        uploaded_by:   "System Auto-Collection (TourAPI)",
        status:        :draft,
        price_per_night: nil
      }
    end
  end
end
```

### 4.3 PropertyImporter (DB 저장)

**파일**: `app/services/property_importer.rb`

```ruby
class PropertyImporter
  Result = Struct.new(:created, :updated, :skipped, :errors, keyword_init: true)

  def initialize(properties_data)
    @data = properties_data
  end

  def import!
    result = Result.new(created: 0, updated: 0, skipped: 0, errors: [])

    @data.each do |attrs|
      import_one(attrs, result)
    end

    result
  end

  private

  def import_one(attrs, result)
    external_id = attrs[:external_id]

    # external_id 기준으로 upsert (중복 방지)
    property = Property.find_or_initialize_by(external_id: external_id)

    if property.new_record?
      property.assign_attributes(attrs)
      if property.save(validate: false)  # price_per_night nil 허용
        result.created += 1
      else
        result.errors << { external_id: external_id, errors: property.errors.full_messages }
      end
    else
      # 이미 존재 → 관리자가 수정한 경우 덮어쓰지 않음
      result.skipped += 1
    end
  rescue => e
    result.errors << { external_id: external_id, error: e.message }
  end
end
```

## 5. Rake Task 설계

**파일**: `lib/tasks/crawl.rake`

```ruby
namespace :crawl do
  desc "한국관광공사 TourAPI로 펜션/숙소 데이터 수집"
  task properties: :environment do
    puts "=== VINTEE 펜션 데이터 수집 시작 ==="
    start_time = Time.current

    # API 키 확인
    unless ENV["TOUR_API_KEY"].present?
      puts "ERROR: TOUR_API_KEY 환경변수가 설정되지 않았습니다."
      puts "발급: https://api.visitkorea.or.kr"
      exit 1
    end

    # 크롤링 실행
    crawler = Crawlers::TourApiCrawler.new
    properties_data = crawler.call

    # DB 저장
    importer = PropertyImporter.new(properties_data)
    result = importer.import!

    # 결과 출력
    elapsed = (Time.current - start_time).round(2)
    puts "\n=== 수집 결과 ==="
    puts "생성: #{result.created}건"
    puts "스킵(중복): #{result.skipped}건"
    puts "에러: #{result.errors.size}건"
    puts "소요 시간: #{elapsed}초"

    if result.errors.any?
      puts "\n에러 목록:"
      result.errors.each { |e| puts "  - #{e}" }
    end

    puts "\n완료! 관리자 대시보드에서 숙소를 검토하세요: /host/dashboard"
  end

  desc "수집된 draft 숙소 통계 확인"
  task stats: :environment do
    total = Property.count
    draft = Property.draft.count
    auto = Property.auto_collected.count
    pending = Property.pending_review.count

    puts "=== 숙소 통계 ==="
    puts "전체: #{total}건"
    puts "자동수집(전체): #{auto}건"
    puts "검토 대기(draft + 자동수집): #{pending}건"
    puts "활성: #{Property.active.count}건"
  end
end
```

### 컨트롤러 액션 추가 (`host/properties_controller.rb`)

```ruby
# PATCH /host/properties/:id/approve
def approve
  @property.update!(status: :active)
  redirect_to host_dashboard_path, notice: "숙소가 승인되었습니다."
end

# DELETE /host/properties/:id/reject
def reject
  @property.destroy!
  redirect_to host_dashboard_path, notice: "숙소가 거부·삭제되었습니다."
end
```

### 라우트 추가 (`config/routes.rb`)

```ruby
namespace :host do
  resources :properties, only: [:index, :new, :create, :edit, :update, :destroy] do
    member do
      patch :approve
      delete :reject
    end
  end
end
```

## 8. 시퀀스 다이어그램

```
bin/rails crawl:properties
    │
    ▼
TourApiCrawler.call
    │── AREA_CODES 순회 (8개 지역)
    │    ├── GET TourAPI (areaBasedList1)
    │    ├── sleep(1)  ← rate limit
    │    └── normalize() → [{external_id, title, location, ...}]
    │
    ▼
PropertyImporter.import!
    │── 각 항목 처리
    │    ├── Property.find_or_initialize_by(external_id:)
    │    ├── 신규 → save(validate: false)
    │    └── 중복 → skip
    │
    ▼
DB: properties 테이블
    (status: draft, source: "uploaded")
    │
    ▼
/host/dashboard (관리자)
    ├── 검토 대기 목록 표시
    ├── [승인] → status: active (공개)
    └── [거부] → destroy (삭제)
```

## 9. 구현 순서 (Do Phase 가이드)

1. **마이그레이션 생성 및 실행** (5분)
2. **Property 모델 업데이트** (5분)

5. **TourApiCrawler 작성** (20분)
6. **PropertyImporter 작성** (15분)
7. **crawl.rake 작성** (10분)
8. **host/properties_controller.rb approve/reject 액션 추가** (10분)
9. **라우트 추가** (5분)
