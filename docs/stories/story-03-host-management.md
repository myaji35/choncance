# US-03: 호스트 숙소 관리

**Story ID**: US-03
**Epic**: Epic 3 - 호스트 관리
**상태**: ✅ 완료 (100%)
**우선순위**: P0 (Critical)
**작성일**: 2026-02-10

---

## User Story

**As a** 농촌 펜션 호스트 (50-60대 귀농인)
**I want to** 간편하게 숙소를 등록하고 예약을 관리하기
**So that** 전화/문자 수기 관리 부담을 줄이고 온라인으로 수익을 창출할 수 있다

---

## 배경 (Background)

농촌 호스트는 **디지털 도구 활용 경험 부족**으로 숙소 홍보와 예약 관리에 어려움을 겪는다. VINTEE는 **직관적인 호스트 대시보드**와 **간편한 등록 프로세스**로 디지털 진입 장벽을 낮춘다.

### 문제점 (Pain Points)
- 개별 마케팅: 블로그, SNS 관리 어려움
- 예약 관리: 전화/문자로 수기 관리 (중복 예약 위험)
- 정산 복잡: 수기 계좌이체 입금 확인 지연
- 젊은 고객 유입 어려움: 온라인 플랫폼 부재

### 기대 효과 (Expected Benefits)
- 예약 관리 시간 70% 절감 (자동화)
- 중복 예약 위험 0% (실시간 캘린더 동기화)
- 월 평균 예약 건수 5건 → 15건 증가 (플랫폼 노출)
- 디지털 수익원 확보 (농촌 경제 활성화)

## 시나리오 (Scenarios)

### Scenario 1: 숙소 최초 등록 (기본 플로우)

**Context**: 귀농 5년차 박영수 씨(55세)는 VINTEE에 펜션을 등록하려 한다.

**Steps**:
1. VINTEE 회원가입 (`/signup`)
   - 카카오 간편 로그인
   - 기본 정보 입력 (이름, 전화번호)
2. 호스트 프로필 등록 (`/host/profile/new`)
   - 사업자 등록번호 입력 (선택)
   - 은행 계좌 정보 (정산용)
   - 호스트 소개: "안녕하세요, 귀농 5년차 박영수입니다..."
   - **"호스트 신청"** 버튼 클릭
3. 관리자 승인 대기 (1-2일)
   - 이메일 알림: "호스트 신청이 접수되었습니다"
4. 승인 완료 후 숙소 등록 시작 (`/host/properties/new`)
5. **기본 정보** 입력:
   - 숙소명: "논뷰 힐링 펜션"
   - 주소: "경기도 양평군 양평읍..."
   - 숙소 유형: 독채 펜션
   - 최대 인원: 4명
   - 침실 수: 2개
   - 욕실 수: 1개
6. **가격 및 정책** 입력:
   - 1박 가격: 80,000원
   - 체크인 시간: 15:00
   - 체크아웃 시간: 11:00
   - 최소 숙박: 1박
   - 취소 정책: "체크인 7일 전까지 전액 환불"
7. **편의시설** 선택 (복수 선택):
   - ✅ Wi-Fi
   - ✅ 주차
   - ✅ BBQ 시설
   - ✅ 취사 가능
   - ✅ 에어컨
   - ❌ 수영장
8. **태그** 선택 (최소 2개, 최대 5개):
   - #논뷰맛집 ✅
   - #개별바베큐 ✅
   - #SNS맛집 ✅
9. **호스트 스토리** 작성 (최대 500자):
   ```
   안녕하세요, 논뷰 힐링 펜션 호스트 박영수입니다.
   저희 펜션은 양평의 넓은 논 한가운데 위치해 있어,
   아침마다 논을 바라보며 커피 한 잔의 여유를 즐기실 수 있습니다.

   귀농 5년차로, 도시 생활의 피로를 씻고 자연에서 힐링하는
   경험을 제공하고자 펜션을 시작했습니다.

   저희 펜션만의 특별함:
   - 넓은 논 전망 (일출/일몰 감상)
   - 조용한 환경 (마을 외곽)
   - 개별 BBQ 시설 (숯/그릴 제공)

   **정직한 불편함 공유**:
   - Wi-Fi가 가끔 불안정할 수 있습니다 (농촌 환경)
   - 여름철 모기가 있을 수 있습니다 (모기향 제공)

   편안한 촌캉스를 위해 최선을 다하겠습니다!
   ```
10. **사진 업로드** (최소 5장, 최대 10장):
    - 대표 사진 (논 전망)
    - 외관 사진
    - 침실 사진 (2장)
    - BBQ 시설 사진
    - 주변 풍경 사진
11. **경험 프로그램** 추가 (선택):
    - 아궁이 체험 (2시간, 30,000원)
    - 농사 체험 (3시간, 40,000원)
12. **미리보기** 확인
    - 숙소 상세 페이지 미리보기
    - 정보 확인 후 수정
13. **"등록하기"** 버튼 클릭
14. 관리자 승인 대기 (1-2일)
    - 이메일 알림: "숙소 등록이 접수되었습니다"
15. 승인 완료 후 자동 게시
    - 이메일 알림: "숙소가 게시되었습니다!"
    - 호스트 대시보드에서 확인 가능

**Expected Outcome**:
- ✅ 총 소요 시간 30분 이내
- ✅ 직관적인 단계별 입력 UI
- ✅ 호스트 스토리로 진정성 전달
- ✅ 정직한 불편함 표현으로 신뢰 구축

### Scenario 2: 예약 승인 및 거부

**Context**: 박영수 씨는 새로운 예약 요청을 받았다.

**Steps**:
1. 호스트 대시보드 접속 (`/host/dashboard`)
2. **"새 예약"** 알림 확인 (1건)
3. 예약 목록 페이지로 이동 (`/host/bookings`)
4. 예약 상세 확인:
   - 게스트: 김지수
   - 체크인: 2026-02-15
   - 체크아웃: 2026-02-16
   - 인원: 2명
   - 총 금액: 84,000원
   - 특별 요청: "조용한 방 부탁드립니다"
5. 캘린더에서 가용성 확인
   - 해당 날짜 예약 가능
6. **"승인하기"** 버튼 클릭
7. 예약 상태 변경: PENDING → CONFIRMED
8. 자동 알림 발송:
   - 게스트: "예약이 승인되었습니다" 이메일
   - 호스트: "예약 승인 완료" 이메일
9. 캘린더 자동 업데이트 (해당 날짜 예약 불가)

**대체 시나리오: 예약 거부**
1. 예약 상세 확인
2. 캘린더에서 중복 예약 발견 (수동 예약)
3. **"거부하기"** 버튼 클릭
4. 거부 사유 입력: "죄송합니다, 해당 날짜는 이미 예약되었습니다"
5. 예약 상태 변경: PENDING → CANCELLED
6. 게스트에게 거부 알림 발송
7. 자동 환불 처리 (결제 취소)

**Expected Outcome**:
- ✅ 실시간 예약 알림
- ✅ 간단한 승인/거부 프로세스
- ✅ 자동 캘린더 동기화

### Scenario 3: 숙소 정보 수정

**Context**: 박영수 씨는 가격을 인상하고 새 사진을 추가하려 한다.

**Steps**:
1. 호스트 대시보드에서 **"내 숙소"** 클릭
2. 숙소 리스트에서 "논뷰 힐링 펜션" 선택
3. **"수정하기"** 버튼 클릭
4. 가격 변경: 80,000원 → 90,000원
5. 새 사진 추가 (3장)
6. 호스트 스토리 업데이트 (계절 정보 추가)
7. **"저장하기"** 버튼 클릭
8. 즉시 반영 (관리자 재승인 불필요)
9. 기존 예약에는 영향 없음 (가격 동결)
10. 신규 예약부터 새 가격 적용

**Expected Outcome**:
- ✅ 즉시 수정 가능
- ✅ 기존 예약 보호
- ✅ 신규 예약만 변경 사항 적용

---

### Scenario 4: 예약 현황 및 수익 확인

**Context**: 박영수 씨는 이번 달 예약 현황을 확인하려 한다.

**Steps**:
1. 호스트 대시보드 접속 (`/host/dashboard`)
2. **KPI 요약** 확인:
   - 이번 달 예약: 12건
   - 이번 달 수익: 1,020,000원 (수수료 차감 후)
   - 다음 달 예약: 8건
   - 평균 평점: 4.7/5.0
3. **예약 캘린더** 확인:
   - 2월 달력 표시
   - 예약된 날짜: 파란색 배경
   - 예약 가능: 흰색 배경
   - 예약 불가 (수동): 회색 배경
4. **최근 예약** 리스트:
   - 예약 1: 김지수, 2/15-2/16, 확정
   - 예약 2: 이민호, 2/22-2/23, 확정
   - 예약 3: 박은지, 3/1-3/2, 대기
5. **수익 내역** 확인:
   - 총 예약 금액: 1,200,000원
   - 호스트 수수료 (15%): -180,000원
   - **순수익: 1,020,000원**
   - 다음 정산일: 3월 5일

**Expected Outcome**:
- ✅ 직관적인 대시보드 UI
- ✅ 실시간 예약 현황 확인
- ✅ 투명한 수익 내역
