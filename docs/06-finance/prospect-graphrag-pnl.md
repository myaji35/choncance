## 5. 손익분기점 (BEP) 분석

### 5.1 월 BEP (고정비 기반)

고정비 구조 (최소):
- 서버 + SMS + 도메인: 270만원
- CS 1명: 350만원
- 사무·법무: 250만원
- **월 최소 고정비: 870만원**

변동비: LLM API + PG 수수료 (GMV·사용량 연동)

**BEP 호스트 수** (Starter + 월 3박 평균 기준):
- 호스트 1명당 기여 이익: 16,000원(구독) + 18,000원(수수료) - 11,760원(PG 수수료) = **22,240원**
- 월 BEP: 870만원 ÷ 22,240원 = **약 390명 유료 호스트**

👉 **Month 6 (420명)에 월 BEP 도달.**

### 5.2 누적 BEP (LLM 선투자 포함)

- Phase 1-5 개발 기간(11주) 동안 LLM 인덱싱·서버·개발 고정비: 약 1억원 선투자 가정
- 누적 BEP: Month 9~10 예상
